# Registro Spese - Full Stack Web App

Applicazione web per la gestione delle finanze personali, sviluppata per implementare un'architettura Full Stack completa.

## Funzionalità
- Gestione Transazioni: Aggiunta e rimozione di entrate e uscite.
- Calcolo Automatico: Aggiornamento in tempo reale del bilancio totale.
- Persistenza Dati: Gestione dei dati tramite database JSON lato server.
- Docker: Applicazione containerizzata per garantire portabilità e facilità di distribuzione.

## Tecnologie Utilizzate
- Frontend: HTML5, CSS3, JavaScript (Fetch API).
- Backend: Node.js, Express.
- Database: File System (JSON).
- Infrastruttura: Docker, Docker Compose.

## Istruzioni per l'avvio
1. Clonare il repository.
2. Eseguire il comando "docker-compose up" nel terminale.
3. Accedere all'indirizzo http://localhost:3000 tramite browser.
